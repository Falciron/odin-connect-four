# odin-connect-four
A terminal implementation of [Connect Four](https://www.theodinproject.com/lessons/ruby-connect-four), written in Ruby, for [The Odin Project](https://www.theodinproject.com/).

Halfway through the Ruby track, this project has taught or reinforced for me:
- Advantages and disadvantages of test driven development
- Authoring advanced unit tests, including mocking, stubbing, and IO overrides
- Recognizing and building toward the minimum requirements of a project, in order to minimize the diversity of test cases
- Adopting naming conventions for instance methods, fostering legibility and coherence within test files

## Pre-requisites for Usage

Ruby 3+

## Instructions for Usage

Type `ruby main.rb` into your terminal to start playing the game.

## Instructions for Testing

1. Type `bundle install` into your terminal to install rspec and rubocop.
2. Type `rspec` into your terminal to run all tests contained within the /spec/ folder.

## Instructions for Static Code Analysis

Type `bundle exec rubocop -a` into your terminal to detect and auto-resolve many issues.
